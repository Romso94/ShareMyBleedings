Comment lire ce dashboard
Légende & guide d'utilisation

Niveaux de risque

CRITICAL score ≥ 85 — HIGH 65–84 — MEDIUM 35–64 — INFO < 35

Le score combine : nature des comptes autorisés (Everyone, Domain Users…), droits effectifs (Full/Change/Read), accès anonyme, signing SMB, et contenu sensible détecté.

Bloc "Qui a accès ?"

Dans la vue Par machine, chaque finding liste les comptes en 3 groupes :

  • › Accès risqué — comptes dangereux (Everyone, Domain Users, Authenticated Users, BUILTIN\Users…)
  • › Autres comptes — comptes nommés / groupes spécifiques
  • › Accès légitime — SYSTEM, BUILTIN\Administrators, CREATOR OWNER

Droits testés

Permissions effectivement vérifiées via une session de test :

  • › LIST lister fichiers/dossiers
  • › READ lire/télécharger les contenus
  • › WRITE créer/modifier/écraser
  • › DELETE supprimer fichiers/dossiers
  • › EXECUTE exécuter programmes/scripts
  • › SPECIAL lire/modifier les ACL

Sensitive content

Tab listing files found by the content scanner (manspider) matching keywords like password, api_key, iban, BEGIN PRIVATE...

Signing SMB

Signing OK signature SMB requise (recommandé). Signing OFF signature désactivée → exposé au relais NTLM.

Actions

  • › Cliquer sur un partage / son chemin pour copier le \\HOST\share
  • › Cliquer sur une commande de remédiation pour la copier
  • › Pills de filtres par niveau + recherche plein-texte
  • › Export CSV / Excel (avec couleurs) / HTML / PDF
  • › Toggle colonnes dans Par risque, marquer "résolu" par finding
